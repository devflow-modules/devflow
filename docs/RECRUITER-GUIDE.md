## What To Check Beyond Code

When reviewing these repositories, consider not only implementation details but also:

- Whether the problem is clearly defined.
- Whether architecture decisions are documented.
- Whether security and privacy trade-offs are explicit.
- Whether tests and setup instructions are visible.
- Whether the project can be understood quickly by a non-author reviewer.
