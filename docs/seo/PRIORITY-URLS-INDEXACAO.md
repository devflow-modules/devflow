# URLs prioritárias para indexação manual

Use esta lista no **Google Search Console** para inspecionar e solicitar indexação das páginas mais importantes.

## Como usar

1. Acesse [Google Search Console](https://search.google.com/search-console) → Inspeção de URL.
2. Cole cada URL abaixo e clique em **Solicitar indexação** (quando disponível).
3. Priorize as 5–10 primeiras se o limite diário for restrito.
